# 🚀 Advanced Multi-LLM Trading System

High-accuracy AI responses through **ensemble voting** of multiple LLMs.

## ✨ Features

- **🎯 Ensemble Method**: Query all LLMs in parallel, get best response via voting
- **⚡ Parallel Method**: Get fastest response from any active LLM  
- **👥 Consensus Method**: Majority voting for maximum confidence
- **📊 Accuracy Scoring**: AI-powered response evaluation (0-100%)
- **💾 Smart Caching**: Reduce API calls and costs
- **🆓 Zero Cost**: Works with 100% free API tiers
- **📱 Mobile Ready**: Optimized for all devices

## 🚀 Quick Start

### 1️⃣ Get Free API Keys

All of these are **100% FREE, no credit card required**:

- **Groq**: https://console.groq.com
- **Gemini**: https://aistudio.google.com  
- **Claude**: https://console.anthropic.com
- **OpenRouter**: https://openrouter.ai

### 2️⃣ Deploy to Netlify (Frontend)

1. Go to https://app.netlify.com
2. Click "New site from Git"
3. Select this repository
4. Deploy (automatic)
5. Add your API keys in the Setup tab

### 3️⃣ Deploy Backend to Railway (Optional)

1. Go to https://railway.app
2. Create new project from GitHub
3. Add environment variables:
GROQ_API_KEY=your_key
GEMINI_API_KEY=your_key
CLAUDE_API_KEY=your_key

4. Deploy automatically
